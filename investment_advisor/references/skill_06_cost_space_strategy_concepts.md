### 核心理念

**投资是反人性的**:
```
高成本持仓的问题：
下跌 → 心理压力大 → 想抛
上涨 → 怕回撤 → 也想抛
结果：拿不住，错失大行情

低成本持仓的优势：
下跌 → 仍有浮盈 → 不慌
上涨 → 利润丰厚 → 淡定
结果：稳坐钓鱼台，收益最大化
```

### 控成本策略

#### 方法1: 主升浪时的操作

**原则**: 趋势正确时，尽量不出

```
场景：大饼/英伟达在主升浪中

❌ 错误做法：
频繁买卖 → 成本不断抬高 → 踏空主升浪

✅ 正确做法：
主力仓位（80%）：持有不动
灵活仓位（20%）：
├─ 拆成4个5%
├─ 波段操作
└─ 目的：降低总成本

示例：
总仓位100股，成本$100
卖出20股@$120（获利$2000）
回调时买回20股@$110
→ 总成本降至$98（100×100-2000+20×110）/100
→ 空间扩大，更容易持有
```

#### 方法2: 波段套利降成本

**波段操作节奏**:
```
持仓配置：
├─ 核心仓位（70%）：长期持有
└─ 机动仓位（30%）：波段操作

波段操作规则：
1. 涨幅达10-15%：卖出5%仓位
2. 回调5-8%：买回5%仓位
3. 重复操作，持续降低成本
4. 核心仓位永不动

成本控制表：
轮次 | 操作 | 价格 | 盈亏 | 累计降低成本
-----|------|------|------|-------------
初始 | 买入 | $100 | -    | $100
第1轮| 卖5% | $115 | +$75 | $99.25
第1轮| 买5% | $108 | -    | $99.25
第2轮| 卖5% | $125 | +$85 | $98.40
第2轮| 买5% | $118 | -    | $98.40

效果：成本从$100降至$98.40，空间扩大
```

### 撑空间策略

#### 心理空间构建

```
低成本持仓的心理优势：

场景1：市场回调-15%
高成本持仓：浮亏，恐慌，想割
低成本持仓：仍有利润，淡定，可加仓

场景2：市场上涨+50%
高成本持仓：利润薄，怕回撤，想卖
低成本持仓：利润厚，有空间，能持有

场景3：市场震荡
高成本持仓：煎熬，频繁操作
低成本持仓：从容，波段套利
```

#### 空间扩大的三个维度

**维度1：价格空间**
```
成本$100 vs 成本$80

价格跌至$85时：
├─ 成本$100：亏损-15%（心理压力大）
└─ 成本$80：盈利+6.25%（仍有浮盈）

价格涨至$120时：
├─ 成本$100：盈利+20%（容易获利了结）
└─ 成本$80：盈利+50%（有空间继续持有）
```

**维度2：时间空间**
```
低成本 = 更长持有时间 = 把握更大行情

高成本：
├─ 等待解套：心理煎熬
├─ 小有盈利：急于兑现
└─ 持有周期：3-6个月

低成本：
├─ 始终有利润：心态平和
├─ 不急于卖出：等待大机会
└─ 持有周期：1-3年甚至更久
```

**维度3：选择空间**
```
低成本带来的选择自由：

1. 进攻选择：
   ├─ 加仓：有资金，有勇气
   └─ 持有：坐等升值

2. 防守选择：
   ├─ 减仓：获利了结
   ├─ 持有：等待更好机会
   └─ 加仓：越跌越买

3. 中立选择：
   └─ 波段操作：套利降成本
```
