### 军队调度建仓法

```
建仓流程三步骤：

第一步：斥候刺探（观察期）
├─ 时间：1-2周
├─ 仓位：0%
├─ 任务：收集信息、观察走势
└─ 输出：初步判断

第二步：先锋开路（试探期）
├─ 时间：1-4周
├─ 仓位：10-20%
├─ 任务：小额建仓、验证判断
└─ 输出：策略确认

第三步：大军跟进（主力期）
├─ 时间：视情况而定
├─ 仓位：50-70%
├─ 任务：主力资金入场
└─ 输出：完成建仓
```

### 仓位管理规则表

| 市场状态 | 个人判断 | 走势验证 | 仓位操作 | 备注 |
|---------|---------|---------|---------|------|
| 上涨趋势 | 看多 | ✅ 确认 | 加仓至70% | 顺势而为 |
| 上涨趋势 | 看空 | ❌ 冲突 | 立即止损 | 我错了 |
| 下跌趋势 | 看空 | ✅ 确认 | 减仓至30% | 保护利润 |
| 下跌趋势 | 看多 | ❌ 冲突 | 立即止损 | 我错了 |
| 震荡区间 | 不确定 | 观察中 | 维持50% | 波段操作 |
| 暴跌 | 基本面健康 | ✅ 确认 | 加仓 | 越跌越买 |

### 加仓时机判断

**满足以下条件时加仓**:
1. ✅ 基本面没有恶化
2. ✅ 市场走势确认我的判断
3. ✅ 价格大幅下跌（机会成本低）
4. ✅ 仓位有预留空间（<85%）

**加仓操作原则**:
```
原则：越跌越买，但要有节奏

示例：某股票从100元跌到80元
├─ 第一次加仓：80元（跌20%）
├─ 第二次加仓：70元（再跌12.5%）
├─ 第三次加仓：60元（再跌14.3%）
└─ 预留资金：应对更大跌幅

注意：每次加仓金额可递增（越跌越重）
```
