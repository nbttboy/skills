#### "第一 + 唯一" 双重标准

```
┌─────────────────────────────────────┐
│  标的质量评估矩阵                    │
├─────────────────────────────────────┤
│                                     │
│  第一（规模）         唯一（壁垒）   │
│  ↓                    ↓             │
│  • 行业龙头           • 核心技术     │
│  • 市占率高           • 创新能力     │
│  • 规模效应           • 稀缺性       │
│  • 品牌效应           • 不可复制     │
│                                     │
│  货币化能力 ←─────→ 价值创造能力     │
│                                     │
└─────────────────────────────────────┘
```

#### 具体判断维度

**规模指标（第一）**:
- 市场占有率 > 20% (行业前三)
- 营收规模持续增长
- 行业话语权和定价权
- 规模经济效应明显

**壁垒指标（唯一）**:
- 技术专利/独家技术
- 品牌护城河（如茅台 vs 杂牌酒）
- 网络效应（用户越多价值越大）
- 转换成本高（客户难以更换）

### 选择决策树

```
开始选择标的
    ↓
是否行业龙头？
    ├─ 否 → 放弃
    └─ 是 → 下一步
         ↓
     是否有护城河？
         ├─ 否 → 谨慎（可能被替代）
         └─ 是 → 下一步
              ↓
         基本面是否健康？
              ├─ 否 → 放弃
              └─ 是 → 下一步
                   ↓
               估值是否合理？
                   ├─ 否 → 等待机会
                   └─ 是 → 可以建仓
```
