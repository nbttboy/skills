### 四大核心要素

#### 要素1: 价格档差设计

**原则**: 档差必须足够大，才能有效降低成本

```
❌ 错误示例：档差过小
买入价1: $100
买入价2: $98   (仅跌2%)  ← 无意义
买入价3: $96   (再跌2%)  ← 浪费仓位
问题：成本仅降低0.67%，浪费资金

✅ 正确示例：档差合理
买入价1: $100
买入价2: $85   (跌15%)   ← 有效
买入价3: $70   (再跌17.6%)  ← 显著降低成本
效果：成本降至$85，降幅15%
```

**档差设计公式**:
```
建议档差 = 当前价格 × (10% ~ 20%)

高波动资产（加密货币）: 15-25%
中波动资产（科技股）: 10-20%
低波动资产（蓝筹股）: 8-15%
```

#### 要素2: 分批买入原则

**口诀**: 不怕踏空，就怕踩坑

```
分批买入计划表：

轮次  | 价格区间      | 仓位占比 | 资金占比 | 累计仓位
-----|--------------|---------|---------|----------
初始  | $100         | 20%     | 20%     | 20%
第一批 | $85 (-15%)  | 25%     | 25%     | 45%
第二批 | $70 (-17.6%)| 30%     | 30%     | 75%
预留  | 应急备用      | -       | 25%     | 保持弹性

原则：
1. 永不梭哈（一次性全进）
2. 越跌越买，但每次金额递增
3. 始终保留25%应急资金
```

**分批买入心理管理**:
```
场景1: 买后继续跌
心态：还好有预留资金，可以继续加仓
行动：按计划执行下一档买入

场景2: 买后立即涨
心态：已有部分仓位，不错过机会
行动：小踏空可接受，保持纪律

场景3: 大幅下跌
心态：基本面健康，机会难得
行动：加大下一档的买入比例
```

#### 要素3: 仓位预留策略

**仓位上限规则**:
```
个人常规配置：
├─ 正常仓位：70-75%
├─ 灵活区间：75-80%
├─ 极限上限：85%（特殊情况）
└─ 禁止满仓：永远不100%

预留资金用途：
1. 应对市场给出的加仓机会
2. 应急资金（生活/突发事件）
3. 心理安全垫（避免强制平仓）
```

**仓位管理表**:

| 市场阶段 | 建议仓位 | 预留资金 | 策略重点 |
|---------|---------|---------|---------|
| 熊市底部 | 75-80% | 20-25% | 大胆加仓 |
| 上升初期 | 70-75% | 25-30% | 稳步建仓 |
| 主升浪 | 65-70% | 30-35% | 持有为主 |
| 高位震荡 | 50-60% | 40-50% | 逐步减仓 |
| 顶部信号 | 30-40% | 60-70% | 大幅减持 |

#### 要素4: 多元化配置

**原则**: 不在单一方向上仓位过重

```
资产配置示例：

单一股票上限：不超过总资产的30%
单一行业上限：不超过总资产的40%
单一资产类别：不超过总资产的60%

示例配置（100万总资产）：
├─ 科技股：40万（40%）
│   ├─ 英伟达：25万
│   └─ 台积电：15万
├─ 加密货币：20万（20%）
│   └─ 比特币：20万
├─ 房产（自住）：30万（30%）
└─ 现金/货基：10万（10%）预留
```

**多元化的三个维度**:
1. **资产类别**: 股票、房产、加密货币、债券等
2. **行业领域**: 科技、消费、医疗、能源等
3. **地理区域**: 国内、美股、港股等

### 抄底执行清单

**抄底前检查**:
- [ ] 基本面是否健康（营收、利润、市占率）？
- [ ] 下跌是否属于非理性（情绪、误杀）？
- [ ] 价格档差是否已设计（>10%）？
- [ ] 仓位是否有预留（<85%）？
- [ ] 是否制定了分批计划？
- [ ] 整体配置是否多元化？

**抄底中监控**:
- [ ] 每次买入是否严格按照价格档差？
- [ ] 是否避免了情绪化梭哈？
- [ ] 是否保持了资金预留？

**抄底后评估**:
- [ ] 成本是否有效降低？
- [ ] 仓位是否在安全范围？
- [ ] 后续是否仍有应对空间？
