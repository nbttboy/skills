### 核心判断维度

#### 维度1：技术层面

**震荡区与博弈区**:
```
价格走势图：

$150 ┤           ╭─震荡区─╮  ← 反复无法突破
     │           │       │
$140 ┤           │       │
     │       ┌───┘       └───┐
$130 ┤   ┌───┘               └──
     │   │
$120 ┤───┘  主升浪阶段
     │
     └────────────────────────→ 时间

判断标准：
1. 在震荡区停留时间 > 主升浪时间的1/3
2. 多次冲击压力位失败（3次以上）
3. 成交量萎缩（相比主升浪期）
4. 波动率下降

结论：趋势可能终结，考虑离场
```

**博弈区位置分析**:
```
关键位置判断：

历史高点：$150
当前位置：$145-150区间震荡
支撑位置：$130

分析：
├─ 距离历史高点：0-3%（很近）
├─ 突破压力：巨大（需要新催化剂）
├─ 下跌空间：13%至支撑位
└─ 风险收益比：不利（下跌空间 > 上涨空间）

决策：逐步减仓
```

#### 维度2：基本面层面

**确定性评估框架**:

```
确定性检查清单：

□ 行业趋势
  ├─ [ ] 行业增速是否持续？
  ├─ [ ] 政策环境是否友好？
  └─ [ ] 市场需求是否旺盛？

□ 公司基本面
  ├─ [ ] 营收增长是否维持？
  ├─ [ ] 利润率是否稳定/提升？
  ├─ [ ] 市占率是否继续扩大？
  └─ [ ] 竞争优势是否加强？

□ 估值水平
  ├─ [ ] PE是否合理？
  ├─ [ ] 是否有泡沫化迹象？
  └─ [ ] 市场预期是否过高？

判断标准：
- 3项以上出现问题：趋势终结，离场
- 1-2项问题 + 技术面恶化：减仓
- 基本面健康 + 技术面震荡：持有观察
```

### 决策矩阵

```
技术面 \ 基本面 | 健康 | 恶化 | 不确定
----------------|------|------|--------
上涨突破        | 持有 | 减仓 | 观察
横盘震荡        | 持有 | 离场 | 减仓
下跌趋势        | 减仓 | 离场 | 离场

示例1：英伟达2024
技术面：主升浪
基本面：AI大势仍在，营收高增长
决策：持有

示例2：某科技股高位
技术面：反复冲高回落
基本面：增速放缓，竞争加剧
决策：离场
```

### 离场策略

**分批离场方案**:

```
离场阶段设计：

第一阶段：试探性减仓（20%）
├─ 触发条件：趋势终结信号出现
├─ 操作：卖出20%仓位
└─ 目的：锁定部分利润，观察后续

第二阶段：主要减仓（40%）
├─ 触发条件：信号确认，开始下跌
├─ 操作：卖出40%仓位
└─ 目的：保护大部分利润

第三阶段：清仓离场（剩余40%）
├─ 触发条件：趋势明确终结
├─ 操作：全部清仓
└─ 目的：完全退出，寻找新机会

注意：每个阶段可根据实际情况调整比例
```
