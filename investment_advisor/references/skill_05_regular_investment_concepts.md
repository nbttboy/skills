### 三大注意事项

#### 注意事项1: 估值判断

**高估值处理**:
```
判断标准：
- PE高于历史均值50%以上
- 持续创新高但基本面未改善
- 市场情绪极度乐观

策略：
├─ 停止定投 或
└─ 仅在暴跌时买入（-10%以上）

示例：当前纳指、标普500
问题：估值过高，缺乏把握
策略：等待大跌机会（-15%以上）再小额买入
```

**低估值处理**:
```
判断标准：
- PE低于历史均值30%以上
- 基本面健康但被错杀
- 市场情绪悲观

策略：
├─ 加大定投金额（1.5-2倍）
└─ 缩短定投周期（月投→周投）

关键：估值 = 基本面 vs PE的偏离度
看重：增速、营收、利润
```

#### 注意事项2: 时机选择

**最佳定投时机**:
```
时机等级表：

S级（必投）：暴跌 >20%
├─ 金额：2-3倍常规金额
├─ 示例：疫情暴跌、雷曼危机
└─ 原则：别人恐惧我贪婪

A级（加投）：大跌 10-20%
├─ 金额：1.5-2倍常规金额
├─ 示例：财报不及预期
└─ 原则：合理回调机会

B级（正常）：小跌 5-10%
├─ 金额：1倍常规金额
├─ 示例：正常波动
└─ 原则：保持纪律

C级（减投）：横盘或小涨
├─ 金额：0.5倍常规金额
├─ 示例：震荡市
└─ 原则：谨慎观望

D级（停投）：高位或暴涨
├─ 金额：0倍（暂停）
├─ 示例：连续大涨后高位
└─ 原则：不追高
```

#### 注意事项3: 与价格档差结合

**定投+档差双策略**:

```
方案设计：

基础设定：
├─ 当前价格：$100
├─ 月定投额：$1000
└─ 目标：优化买入成本

档差定投表：
价格区间    | 投入金额  | 触发条件       | 仓位占比
-----------|----------|---------------|----------
$95-100    | $500     | 每月固定      | 正常
$85-95     | $1000    | 跌破$95       | 加大
$75-85     | $1500    | 跌破$85       | 大幅加大
$65-75     | $2000    | 跌破$75       | 全力加仓
<$65       | $2500    | 跌破$65       | 极限加仓

核心逻辑：
1. 永远保持定投纪律
2. 越跌越买，金额递增
3. 档差够大，成本有效降低
4. 有心理预期，不慌张
```
